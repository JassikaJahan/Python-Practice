### Problem: Square of side 'N'

**Problem Description:**

You are given an integer `n`. Your task is to return a square pattern of size `n x n` made up of the character '*', represented as a list of strings.

---

**Input:**

- An integer `n` (1 ≤ n ≤ 50): The size of the square (number of rows and columns).

---

**Output:**

- A list of strings where each string is a row of `n` characters, all being `*`.

---

**Example 1:**

Input: 
3
['', '', '***']

---

**Example 2:**

**Input:**

---
5

**Output:**
['', '', '', '', '*****']
